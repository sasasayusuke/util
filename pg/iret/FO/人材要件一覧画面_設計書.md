## 機能概要

SFで登録した案件に紐づく人材要件の一覧を表示する画面。選択して詳細画面へ遷移することができる。

## 画面デザイン

※詳細な画面デザインは別途デザインファイルを参照

## 利用可能権限

| ロール | 条件 |
|--------|------|
| APT | 全機能利用可能 |
| 法人ソルバー | 自社に関連する人材要件のみ参照可能 |
| 個人ソルバー | アクセス不可 |

## イベント定義

| イベント | 処理概要 | 異常時の処理 | 備考 |
|----------|----------|--------------|------|
| 画面初期表示 | 人材要件一覧をAPIから取得し表示 | エラーメッセージを画面上部に表示 | ローディング表示あり |
| 人材要件Noクリック | 選択した要件の詳細画面へ遷移 | - | 人材要件IDをパラメータで渡す |
| 検索ボタンクリック | 入力条件で一覧を絞り込み | 該当データなしメッセージ表示 | |
| ページ切り替え | 指定ページのデータを取得・表示 | エラーメッセージを画面上部に表示 | |

## シーケンス・フロー

1. ユーザーがメニューから「人材要件」を選択
2. 人材要件一覧画面へ遷移
3. 初期表示処理
   - ローディング表示
   - APIコール
   - 一覧データ取得
   - 画面に一覧を表示
4. 検索実行時
   - 検索条件を入力
   - 検索ボタンクリック
   - APIコール（検索条件付き）
   - 結果を画面に表示
5. 詳細画面への遷移
   - 人材要件IDを持って詳細画面へ遷移
