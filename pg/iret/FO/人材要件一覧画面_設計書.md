# 人材要件一覧画面 設計書

## 関連課題

* なし

## 機能概要

Salesforceで登録した案件に紐づく人材要件の一覧を表示する画面。
ユーザーは一覧から人材要件を選択して詳細画面へ遷移でき、検索機能を使用して特定の要件を絞り込むことができる。

## リンク・ボタン項目

| ボタン・リンク名 | 項目種別 | 遷移先 | 初期状態（活性・非活性） | 備考 |
|------------------|----------|--------|-------------------------|------|
| 人材要件No | リンク | 人材要件詳細画面 | 活性 | 各行に表示 |
| 絞り込み | ボタン | - | 活性 | 検索実行 |

## イベント定義

| イベント | 処理概要 | 異常時の処理 | 備考 |
|----------|----------|--------------|------|
| 画面初期表示 | 人材要件一覧をAPIから取得し表示 | エラーメッセージを画面上部に表示 | ローディング表示あり |
| 人材要件Noクリック | 選択した要件の詳細画面へ遷移 | - | 人材要件IDをパラメータで渡す |
| 絞り込みボタンクリック | 入力条件で一覧を絞り込み | 該当データなしメッセージ表示 | |
| リセットボタンクリック | 検索条件をクリアして全件表示 | エラーメッセージを画面上部に表示 | |
| ページ切り替え | 指定ページのデータを取得・表示 | エラーメッセージを画面上部に表示 | |

## シーケンス・フロー

1. ユーザーがメニューから「人材要件」を選択
2. 人材要件一覧画面へ遷移
3. 初期表示処理
   - ローディング表示
   - APIコール
   - 一覧データ取得
   - 画面に一覧を表示
4. 検索実行時
   - 検索条件を入力
   - 検索ボタンクリック
   - APIコール（検索条件付き）
   - 結果を画面に表示
5. 詳細画面への遷移
   - 人材要件Noをクリック
   - 人材要件IDをパラメータに詳細画面へ遷移
