# 人材要件機能フローチャート

## 📋 概要
法人ソルバーが人材要件を確認して、個人ソルバーを申請・管理するシンプルなフローです。

---

## 📊 フローチャート

```mermaid
flowchart TD
    Start([開始: 法人ソルバー]) --> Menu[ソルバーメニュー画面<br/>GUI: 法人ソルバー選択]
    Menu --> Check[人材要件確認<br/>GUI: 「個人ソルバー一覧」クリック]
    Check --> List[個人ソルバー一覧表示<br/>GUI: 検索・フィルタ使用]
    List --> Found{必要な人材<br/>見つかった？}
    
    Found -->|Yes| Select[個人ソルバー選択<br/>GUI: 行クリック]
    Found -->|No| NewApp[新規申請<br/>GUI: 「新規個人ソルバー申請」クリック]
    
    Select --> Detail[詳細画面表示<br/>GUI: 全項目確認]
    Detail --> Suitable{要件に<br/>適合？}
    Suitable -->|Yes| Use[利用決定]
    Suitable -->|No| List
    
    NewApp --> Form[申請フォーム入力<br/>GUI: 32項目入力]
    Form --> Confirm[確認画面<br/>GUI: 「確認画面に進む」クリック]
    Confirm --> Submit[申請完了<br/>GUI: 「申請する」クリック]
    Submit --> Wait[APT承認待ち]
    
    Wait --> Approved{APT承認？}
    Approved -->|Yes| Certified[認定完了]
    Approved -->|No| Reject[申請却下]
    
    Use --> UpdateRate[稼働率・単価更新<br/>GUI: 「稼働率・単価更新」クリック]
    UpdateRate --> BulkEdit[一括編集<br/>GUI: 稼働率・単価入力]
    BulkEdit --> Save[保存<br/>GUI: 「一括保存」クリック]
    Save --> End([終了])
    
    Certified --> UpdateRate
    Reject --> End
    
    style Start fill:#e1f5fe
    style End fill:#c8e6c9
    style Wait fill:#fff3e0
    style Approved fill:#fff3e0
```

## 🔄 時系列の操作フロー

### 1. 法人ソルバー：人材要件確認
**画面**: ソルバーメニュー → 個人ソルバー一覧
**GUI操作**: 「個人ソルバー一覧」ボタンクリック

### 2. 法人ソルバー：既存人材検索
**画面**: 個人ソルバー一覧
**GUI操作**: 検索フィルタ（名前・性別・稼働状況）で絞り込み

### 3-A. 適合人材あり：選択・確認
**画面**: 個人ソルバー詳細
**GUI操作**: 対象行クリック → 詳細確認

### 3-B. 適合人材なし：新規申請
**画面**: 新規個人ソルバー申請
**GUI操作**: 「新規個人ソルバー申請」ボタンクリック → 32項目入力 → 「確認画面に進む」 → 「申請する」

### 4. APT：申請審査・承認
**画面**: 管理画面（APT専用）
**GUI操作**: 申請内容確認 → 承認/却下処理

### 5. 法人ソルバー：稼働率・単価更新
**画面**: 稼働率・単価更新
**GUI操作**: 「稼働率・単価更新」ボタンクリック → 一括編集 → 「一括保存」
