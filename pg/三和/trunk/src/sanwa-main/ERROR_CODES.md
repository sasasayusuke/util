# エラーコード一覧

## エラーコード形式
`<カテゴリ>-<グローバル連番>`

## エラーコード適用方針
### エラーコードを付けるもの
- **予期せぬエラー**: システムが想定していないエラー
- **console.error**: 開発者向けエラーログ
- **システムエラー**: 認証エラー、データベースエラー、API通信エラーなど

### エラーコードを付けないもの
- **明確なバリデーションメッセージ**: 「検索文字を入力してください」「必須項目です」など
- **業務ルールメッセージ**: ユーザーが内容を理解できる明確なメッセージ

## カテゴリ一覧
- **AUTH**: 認証・認可まわり
- **DB**: データベース関連
- **API**: 外部/内部API通信
- **UI**: フロントエンド表示や操作
- **SYS**: サーバー・環境依存（設定、リソース不足など）
- **GEN**: その他・予期せぬエラー

## エラーコード定義（グローバル連番）

### AUTH - 認証・認可
- AUTH-001: 権限不足
- AUTH-002: トークン無効 / 設定された権限が不正
- AUTH-003: ユーザーが見つかりません / 権限登録が設定されていない

### DB - データベース関連
- DB-004: レコードが見つかりません / 表示できる明細データがありません / 該当データは存在しません
- DB-005: データ保存失敗 / 更新済みの為、修正できません
- DB-006: ロック取得失敗 / 検収処理済みデータです / ロック処理でエラー
- DB-077: SQLでエラーが発生しました (21_検索ダイアログテンプレート.js)
- DB-078: ストアドプロシージャでエラーが発生しました (21_検索ダイアログテンプレート.js)

### API - API通信
- API-007: API接続エラー / Error fetching tax
- API-008: 不正なAPIレスポンス / APIエラーメッセージ
- API-017: API errorData (売上明細入力)
- API-018: API errorData (仕入明細入力)
- API-065: API errorData (22_ダイアログ共通)
- API-075: API throw error (売上明細入力 初期化)
- API-076: API throw error (売上明細入力 削除)

### UI - フロントエンド
